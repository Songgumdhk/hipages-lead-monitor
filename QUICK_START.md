# Quick Start Guide

## Installation (5 minutes)

### 1. Create Icons
- Open `create-icons.html` in your browser
- Click "Generate Icons"
- Right-click each canvas and save as `icon16.png`, `icon48.png`, `icon128.png`
- Place them in the extension folder

### 2. Load Extension
1. Open Chrome → `chrome://extensions/`
2. Enable "Developer mode" (top-right toggle)
3. Click "Load unpacked"
4. Select the extension folder

### 3. Start Using
1. Go to: `https://tradiecore.hipages.com.au/leads`
2. Extension starts automatically
3. Click extension icon to see popup

## First Time Setup

1. **Open Hipages Leads Page**
   - Navigate to the Hipages leads page
   - Wait 2-3 seconds for extension to initialize

2. **Configure Refresh Interval**
   - Click extension icon
   - Set refresh interval (default: 5 seconds)
   - Click "Save"

3. **Monitor Leads**
   - New leads will be highlighted automatically
   - Green border + "NEW" badge appears
   - Check popup for lead count

## Troubleshooting

**Extension not working?**
- Make sure you're on `tradiecore.hipages.com.au/leads`
- Refresh the page
- Check extension is enabled in `chrome://extensions/`

**No leads detected?**
- Verify there are "Rental Bond Cleaning" leads on the page
- Check leads have "Waitlist" or "Available" status
- Open popup to see detected lead count

**Icons missing?**
- Extension works without icons, but Chrome shows default icon
- Use `create-icons.html` to generate icons
- Or use any 16x16, 48x48, 128x128 PNG images
