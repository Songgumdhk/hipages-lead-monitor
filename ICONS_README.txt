ICON FILES REQUIRED
===================

This extension requires three icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

CREATING ICONS
==============

Option 1: Use the Icon Generator
---------------------------------
1. Open create-icons.html in your browser
2. Click "Generate Icons"
3. Right-click each canvas and save as:
   - icon16.png
   - icon48.png
   - icon128.png

Option 2: Create Simple Icons Manually
---------------------------------------
You can create simple colored square icons with:
- Background: Blue gradient (#4a90e2 to #357abd)
- Text: "HL" in white, centered
- Border: Subtle white border

Option 3: Use Online Icon Generator
-----------------------------------
Visit any online icon generator and create:
- 16x16, 48x48, and 128x128 pixel PNG images
- Use any design you prefer
- Save as icon16.png, icon48.png, icon128.png

Option 4: Use Placeholder Images
----------------------------------
For testing, you can use any PNG images temporarily.
The extension will work without proper icons, but Chrome
may show a default icon instead.

PLACEMENT
---------
Place all three icon files in the same directory as:
- manifest.json
- content.js
- popup.html
- etc.
